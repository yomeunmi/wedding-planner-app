# 개인정보처리방침

**최종 업데이트: 2025년 11월 20일**

## 1. 개요

웨딩플래너("본 앱")는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 본 앱이 사용자 정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

## 2. 수집하는 정보

**본 앱은 사용자의 개인정보를 수집하지 않습니다.**

### 2.1 로컬 저장 데이터

본 앱은 다음 데이터를 **사용자의 기기에만 로컬로** 저장합니다:

- 결혼식 날짜
- 준비 시작 날짜
- 체크리스트 항목
- 웨딩홀/드레스샵 투어 정보
- 사용자가 촬영하거나 선택한 사진
- 메모 및 노트

**중요**: 이 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

### 2.2 수집하지 않는 정보

본 앱은 다음 정보를 수집하지 않습니다:

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 위치 정보
- 연락처
- 사용 통계 또는 분석 데이터
- 광고 식별자
- 기기 정보

## 3. 정보의 사용

### 3.1 로컬 데이터 사용 목적

기기에 저장된 데이터는 오직 다음 목적으로만 사용됩니다:

- 결혼 준비 일정 표시
- 체크리스트 관리
- 웨딩홀/드레스샵 정보 저장 및 표시
- 사진 스크랩 관리

### 3.2 데이터 전송

**본 앱은 어떠한 데이터도 외부 서버로 전송하지 않습니다.**

- 인터넷 연결 없이도 모든 기능 사용 가능
- 서버 통신 없음
- 제3자와 데이터 공유 없음

## 4. 정보의 보관 및 보호

### 4.1 데이터 보관

- 모든 데이터는 사용자의 기기에 안전하게 저장됩니다
- iOS: 앱의 샌드박스 환경에 저장
- Android: 앱의 내부 저장소에 저장

### 4.2 데이터 보안

- 기기의 운영체제 수준 보안 적용
- 외부 접근 불가
- 암호화된 저장소 사용 (운영체제 제공)

### 4.3 데이터 백업

- iOS: iCloud 백업에 포함될 수 있음 (사용자 설정에 따름)
- Android: Google Drive 백업에 포함될 수 있음 (사용자 설정에 따름)

**참고**: 백업은 Apple 및 Google의 개인정보처리방침을 따릅니다.

## 5. 데이터 삭제

### 5.1 앱 삭제 시

앱을 삭제하면 모든 로컬 데이터가 완전히 삭제됩니다.

### 5.2 수동 삭제

사용자는 언제든지 앱 내에서 다음을 삭제할 수 있습니다:

- 개별 체크리스트 항목
- 웨딩홀/드레스샵 정보
- 저장된 사진
- 모든 데이터 (앱 설정에서)

## 6. 제3자 서비스

### 6.1 사용하지 않는 서비스

본 앱은 다음 서비스를 사용하지 않습니다:

- 분석 도구 (Google Analytics 등)
- 광고 네트워크
- 소셜 미디어 SDK
- 클라우드 저장소

### 6.2 운영체제 제공 기능

본 앱은 다음 운영체제 기본 기능만 사용합니다:

- 카메라 (사진 촬영용)
- 사진 라이브러리 (사진 선택용)
- 로컬 저장소 (데이터 저장용)

## 7. 아동의 개인정보 보호

본 앱은 만 18세 이상을 대상으로 합니다. 본 앱은 의도적으로 만 18세 미만 아동의 정보를 수집하지 않습니다.

## 8. 권한 사용

### 8.1 iOS 권한

- **카메라**: 웨딩홀/드레스 사진 촬영
- **사진 라이브러리**: 저장된 사진 선택

### 8.2 Android 권한

- **CAMERA**: 웨딩홀/드레스 사진 촬영
- **READ_EXTERNAL_STORAGE**: 저장된 사진 선택
- **WRITE_EXTERNAL_STORAGE**: 사진 저장 (Android 10 이하)

**중요**: 모든 권한은 선택적이며, 거부해도 앱의 기본 기능(일정 관리, 체크리스트)은 사용 가능합니다.

## 11. 사용자의 권리

사용자는 다음 권리를 가집니다:

- 앱 내에서 모든 데이터를 자유롭게 확인 가능

### 11.2 데이터 수정
- 앱 내에서 모든 데이터를 자유롭게 수정 가능

## 12. 데이터 처리 원칙

본 앱은 다음 원칙을 따릅니다:

✅ **최소 수집**: 필요한 최소한의 데이터만 로컬 저장
✅ **투명성**: 모든 데이터 사용 목적 명시
✅ **사용자 제어**: 사용자가 모든 데이터 완전 통제
✅ **보안**: 기기 수준 보안 적용
✅ **로컬 우선**: 서버 전송 없음, 완전한 오프라인 작동
