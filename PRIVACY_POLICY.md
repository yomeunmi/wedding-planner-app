# 개인정보처리방침

**웨딩플래너** 앱의 개인정보처리방침입니다.

## 개요

웨딩플래너는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 앱은 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다.

## 수집하는 정보

**없음**

웨딩플래너 앱은 다음과 같은 개인정보를 수집하지 않습니다:
- 이름, 이메일, 전화번호 등 개인 식별 정보
- 위치 정보
- 사진 또는 미디어
- 연락처
- 사용 패턴 또는 분석 데이터

## 데이터 저장

모든 데이터는 **사용자의 기기에만 로컬로 저장**됩니다:
- 결혼식 날짜 정보
- 준비 일정 정보
- 체크리스트 완료 상태
- 닉네임 (선택사항)

이러한 데이터는:
- 사용자의 기기를 벗어나지 않습니다
- 서버로 전송되지 않습니다
- 제3자와 공유되지 않습니다
- 앱을 삭제하면 완전히 제거됩니다

## 인터넷 연결

본 앱은 **인터넷 연결 없이도 완전히 작동**합니다. 인터넷 연결이 필요한 기능은 없습니다.

## 제3자 서비스

본 앱은 다음과 같은 제3자 서비스를 사용하지 않습니다:
- 광고 네트워크
- 분석 도구
- 소셜 미디어 통합
- 결제 처리

## 아동의 개인정보

본 앱은 13세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다. 실제로 모든 연령대로부터 개인정보를 수집하지 않습니다.

## 개인정보처리방침 변경

개인정보처리방침이 변경될 경우, 앱 업데이트를 통해 공지됩니다. 중요한 변경사항이 있을 경우 앱 내에서 별도로 알림을 제공합니다.
